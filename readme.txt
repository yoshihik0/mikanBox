🍊mikanBox　インストール

1. 「mikanBox」フォルダと「index.php」をサーバーにアップロード
2. 「mikanBox/admin.php」にアクセスすると管理者パスワードを設定する画面になるので
　　パスワードを設定してください。

以上で完了です。

この際、サーバー上のindex.phpと同じディレクトリに、「.htaccess」ファイルが作成されます。
「mikanBox」フォルダは、フォルダ名を変更することもできます。
フォルダ名を変更した場合は、「index.php」上部の「$core_dir」も同じ名前に変更してください。
